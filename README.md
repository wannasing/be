beJs
=============
######这是一个选择器，你可以像CSS一样的去选择元素

它支持CSS2以及CSS3的所有语法，什么？你还不会CSS3？？那你只能玩它最基本的选择方式了。

-------------

######语法说明：be( '这里与CSS一样' );

元素选择器

    be( 'div' );

类选择器

    be( '.div' );
    
ID选择器

    be( '#id' );

后代选择器

    be( 'ul li' );
    be( '.box h4' );
    be( '#box div' );
    be( 'div.box' );
	be( 'li#list' );
